# Future Ideas & Improvements

## Features

- [ ] **Multi-Currency Support**: Allow transactions in different currencies with auto-conversion based on daily rates.
- [ ] **Bill Scanning**: Upload a photo of a receipt or bill and use OCR/AI to auto-populate transaction details.
- [ ] **Budget Rollover**: Option to roll over unused budget amounts to the next month.
- [ ] **Shared Budgets**: Allow multiple users (e.g., family members) to share and contribute to a budget.
- [ ] **Subscription Manager**: Dedicated view to track and manage subscriptions (Netflix, Spotify, etc.) with renewal alerts.
- [ ] **Dark Mode Toggle**: Explicit toggle for dark/light mode in settings (currently system based).

## Technical

- [ ] **PWA Support**: Make the app installable as a Progressive Web App.
- [ ] **Offline Mode**: Cache data locally to allow viewing and basic editing while offline, syncing when back online.
- [ ] **End-to-End Encryption**: Encrypt sensitive financial data on the client side before sending to the backend.
- [ ] **Performance Monitoring**: Integrate a tool like Sentry or LogRocket for real-time error tracking and performance monitoring.
